Centroidal Voronoi Tessellation Using d3.js

This example illustrates the generation of a centroidal Voronoi tessellation with a constant density function.

In geometry, a centroidal Voronoi tessellation (CVT) is a special type of Voronoi tessellation or Voronoi diagrams. A Voronoi tessellation is called centroidal when the generating point of each Voronoi cell is also its mean (center of mass). It can be viewed as an optimal partition corresponding to an optimal distribution of generators. -Wikipedia

Algorithm Pseudocode

	generators = array of N points within the enclosing polygon
	maxIterations = X
	iterations = 0

	while(iterations < maxIterations)
		polygons = generateVoronoiTessellation(generators)
		foreach polygon in polygons
			// Update the generator for that polygon
			generators[polygon.generator] = polygon.centroid
		iterations++
